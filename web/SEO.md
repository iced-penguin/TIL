# SEO robots.txt, sitemap.txt

## robots.txt

クローラー：検索エンジンが情報収集するためにインターネット上を自動巡回するプログラム

robots.txt：

- クローラーへのアクセス許可・不許可を記述するファイル
- ブラックリスト方式

robots.txt以外にも以下のような記述方法もある

- HTMLのmetaタグ
- HTTP X-Robots-Tagヘッダー

## sitemap.xml

サイトマップ：

- Webサイトのページ一覧とそのメタデータを記述
- ファイル形式：XML、テキストファイル、RSS、Atom
- ホワイトリスト方式
