# HTTPヘッダー

## HTTPヘッダーの種類

### User-Agent

ユーザーエージェント：ブラウザ、スクレイピングするボット、ダウンロードマネージャーなど

User-Agentヘッダー（UA）：サーバー側にユーザーエージェントの情報を送る
