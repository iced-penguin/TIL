# dockerコマンド

## docker compose

### up

* イメージが存在しなければイメージ作成
* コンテナ作成
* コンテナ起動


オプション

* -d: バックグラウンドで実行
* —build: ビルドを実行

Dockerfileを更新した場合は—buildを付けないと既存のイメージが使用されるため、反映されない
